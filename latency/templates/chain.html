What is the ThunderMeter?

The Thundermeter was a tool built by Pierre and Olshansky. The aim of the tool is to track different RPC providers' latency and response times to watch POKT become the fastest RPC provider ;).

Technical Details

Every benchmark is created by sending 100 eth_getBalance("0x0000000000000000000000000000000000000000") in parallel, and measuring the latency of each request. This process is repeated 5 times per benchmark with a 5 second delay between. The server making these requests is in Miami, Florida
